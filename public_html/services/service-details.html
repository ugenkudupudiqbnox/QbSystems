Pressbooks LTI 1.3 Integration

Our custom Pressbooks LTI 1.3 plugin enables seamless integration between Moodle and Pressbooks, allowing educators to embed rich, interactive textbooks directly into their courses. This open-source solution facilitates single sign-on and grade passback capabilities.

Key Features
  • LTI 1.3 Standard: Full compliance with the latest LTI security and integration standards.
  • Grade Synchronization: Automatic grade passback from Pressbooks to Moodle gradebook.
  • Single Sign-On: Students access Pressbooks content without separate login credentials.
  • Content Protection: Secure access control ensures only enrolled students can access course materials.
  • Easy Configuration: Simplified setup process for instructors and administrators.
